# yg
YAML template generator

## Development

### Prerequisites
- Go 1.19 or later

### Setup
```bash
# Clone the repository
git clone https://github.com/daylight55/yg.git
cd yg

# Initialize Go module (if not already done)
go mod init

# Install dependencies
go mod tidy
```

### Build and Run
```bash
# Build the project
go build

# Run the application
go run .

# Run tests
go test ./...
```
